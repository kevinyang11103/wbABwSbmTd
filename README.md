## 功能模块

### 系统角色

#### 管理员

- **药品管理**: 管理医院内药品信息，包括添加、修改药品名称及备注等。
- **科室管理**: 管理医院科室信息，支持新增和修改科室名称。
- **医生管理**: 管理医生信息，包括修改个人资料和删除离职医生信息。
- **患者管理**: 维护患者注册信息。

#### 医生

- **预约管理**: 审核患者在线挂号申请。
- **病历管理**: 撰写和记录患者病情。
- **住院管理**: 查看和记录患者的住院治疗情况。

#### 患者

- **在线挂号**: 实现在线选择科室和挂号申请。
- **住院信息查看**: 查看个人住院信息。
- **病历查看**: 查阅过往病历并进行病情跟踪。
